# Media folder

Drop your real files here using these exact names (edit them in
`src/data/config.ts` if you'd rather rename anything):

- `birthday-video.mp4` — the single cinematic video
- `birthday-song.mp3` — background music, starts after first tap/click
- `hero.jpg`
- `school-1.jpg`
- `school-2.jpg`
- `motion-1.jpg`
- `motion-2.jpg`
- `parul-1.jpg`
- `parul-2.jpg`
- `memory-1.jpg` through `memory-5.jpg`

You can use anywhere from 8 to 15 photos — add or remove entries in
`siteConfig.media.photos` (config.ts) and in the photo/memory arrays in
`src/data/content.ts` to match.

Until a file is added, that spot on the page will show a soft placeholder
frame with the expected filename printed on it, so the layout still looks
intentional while you're gathering assets.
